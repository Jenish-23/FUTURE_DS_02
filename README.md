# FUTURE_DS_02 - Customer Churn Analysis Dashboard

🔹 Project Overview

This project focuses on analyzing telecom customer data to identify churn patterns and understand the key factors affecting customer retention. An interactive Power BI dashboard is created to visualize insights and support business decision-making.



🔹 Objectives

* Analyze customer churn behavior
* Identify high-risk customer segments
* Understand factors influencing churn
* Provide actionable business recommendations

🔹 Key Insights

* 📉 Overall churn rate is approximately **27%**
* 📅 Customers with **month-to-month contracts** have the highest churn rate (~43%)
* 📈 Customers with **two-year contracts** show the lowest churn (~3%)
* 👶 **New customers (low tenure)** are more likely to churn
* 💳 Certain payment methods (e.g., electronic check) show higher churn tendencies


🔹 Dashboard Features

* 📌 KPI Cards:

  * Total Customers
  * Churned Customers
  * Churn Rate (%)
  * Average Tenure

* 📊 Visualizations:

  * Churn Distribution (Pie Chart)
  * Churn Rate by Contract Type
  * Tenure vs Churn Analysis
  * Customer Segmentation Analysis

* 🎛️ Interactive Filters (Slicers):

  * Gender
  * Contract
  * Internet Service

🔹 Business Recommendations

* Enhance customer engagement strategies

This project demonstrates how data visualization and analysis can help businesses understand customer behavior and take proactive steps to reduce churn and improve retention.
